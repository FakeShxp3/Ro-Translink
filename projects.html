Core Technologies

The foundational systems that power the Ro-Translink experience

Project Polaris

Project Polaris was P2P's first ever project. Anything bus-chassis related falls directly under Polaris - e.g Gen2.1. We have a few buses that currently operate on Gen2.1, the latest update to our chassis: Xcelsior, Orion V, Enviro500, D(E)60LFR, and our D40LF.

Project Polaris stands out as one of the only bus chassis systems in North America featuring an interactive system, realistic engine operations, smooth suspension, and more. Gen2.1 is also incredibly stable and optimized.

Interactive Systems Realistic Engine Operation Smooth Suspension Highly Optimized

Project Sentinel

Project Sentinel is arguably the most technologically advanced and revolutionary product P2P has made. Sentinel includes anything "automation" related. AI Passengers fall under this category, and so does our TMAC!

The underlying system that powers our TMAC includes:

  • A route assembly and managing system that uses purely math and logic to manage stops
  • A lateness system that uses real-life blocks and scheduling
AI Passengers Route Management Scheduling System TMAC Integration

Project Odyssey

Project Odyssey is the codename for P2P's train chassis. While development is on the backburner until Ro-Translink is fully released with buses, development will pick up when we have enough skytrain stations.

Project Odyssey will be embedded fully into Ro-Translink, making it a multi-operational game. We will have the following trains, running on a hybrid automatic/manual system:

  • Mark 1
  • Mark 2.0/2.5
  • Mark 3
  • Mark 5 (V)
SkyTrain Integration Hybrid Control System Multiple Train Models

Project Spectre

Project Spectre is our cutting-edge experimental initiative focusing on next-generation transit simulation technologies. This project integrates advanced rendering techniques, physics simulation, and AI systems to create the most realistic transit experience possible.

Currently under active development, Spectre will power future versions of our vehicle systems with enhanced visuals, improved performance, and more immersive features.
